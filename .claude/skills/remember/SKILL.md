# Remember

Capture the thing durably so it isn't lost.

## Do this
1. Read `memory/remember.md`.
2. Figure out the **type** of item:
   - **Standing** (repeats: daily / every Monday / etc.) → add under `## Standing`.
   - **One-off / dated** (an appointment, errand, a date to act on) → add under `## Upcoming / one-off` with the date.
   - **A promise to my wife** → add under `## Promises I've made to her`.
   - If it's a pregnancy appointment or due date, also update `context/pregnancy.md`.
3. Append using the format: `- [ ] (when) <thing> — added YYYY-MM-DD` (run `date +%Y-%m-%d` for today).
4. Confirm back in one line what you stored and when it will resurface in check-ins.

Keep it frictionless: if João is vague on timing, make a sensible assumption, store it, and say what you assumed so he can correct it.
